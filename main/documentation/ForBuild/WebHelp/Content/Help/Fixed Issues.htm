Previous features and fixed issues

Idera strives to ensure our products provide quality solutions for your SQL Server needs. This version of the SQL admin toolset includes many improvements, including these fixed issues.

1.6 features

SQL Server 2012 support

All tools included in the SQL admin toolset fully support SQL Server 2012 RTM.

1.5 features

New Linked Server Copy tool

The new Linked Server Copy tool lets you quickly and easily copy linked servers from one SQL Server instance to another. This copy includes all of the associated settings, logins, passwords, and permissions.

New options for the Quick Re-Index tool

With the SQL admin toolset, you can now:

  • Specify the fill factor you want to use when rebuilding a selected index.
  • Enable online processing during a rebuild.
  • Use the Maxdrop setting to specify the maximum degree of parallelism used during the rebuild.

1.5 fixed issues

  • Index Analyzer now lists the columns of an index in the order that they were defined.
  • Patch Analyzer now includes the latest SQL Server service packs.
  • Multi Query now correctly sorts the returned results.
  • Multi Query now correctly supports multi-line strings.
  • Space Analyzer now reports the correct value for the Maximum Potential Size metric.

1.3 fixed issues

Job Editor

You can now successfully retrieve job information from a Server Group that contains duplicate SQL Server instances. (1449)

Job Mover

You can now successfully move SQL Server jobs whose names contain single quotation marks ('). (1469)

Multi Query

You can now successfully clear the returned results of your query from the Query Execution and Results pane.(1470)

Server Groups

Special characters are no longer supported when naming new SQL Server groups. Existing Server Groups, previously created by SQL admin toolset version 1.2 or earlier, that contain special characters are still supported. (1465)

SQL Search

You can successfully search for text strings in a SQL Server job, even if the MSDB system database is not listed in your search criteria. (1490)

1.2 fixed issues

General

  • This release offers full support for SQL Server 2008.
  • You now may browse and select databases on a single server. You must manually add multiple databases on server groups.
  • All server browse lists include any servers defined in server groups.
  • You now can copy error reports to the clipboard.

Connection Check

The Windows Authentication field allows you to run a connection test under an alternate user account.

Database Configuration

Database Configuration includes a new comparison report.

Database Mover

  • Database Mover now supports database clusters.
  • When moving a database, you can set the default for all of the migrated users as the matching databases on the destination server.
  • When you move a database, the database owner also is copied to the destination server.
  • When selecting a destination, you now can browse to servers on different drives.
  • You now can rename a database and then copy it to the same server instance.

Index Analysis

You can select individual or multiple databases before running the Index Analyzer.

Inventory Reporter

The Server Inventory now includes the following fields along with a list of enabled SQL Server Network Libraries:

  • TCP Port Setting
  • Memory Allocation Type
  • Minimum SQL Server Memory
  • Maximum SQL Server Memory
  • Allocated CPU Count
  • Allocated CPU Count I/O
  • Number of Physical Processors
  • Number of Logical Processors

Job Mover

Moving jobs containing multiple steps no longer causes an error.

Partition Generator

Viewing details of a partitioned table no longer generates an error message.

Password Checker

  • You can export results as a TXT, CSV, or XML file.
  • Results show the servers and number of logins tested, bad passwords, blank passwords, and passwords that match the login.
  • The default list now includes additional password dictionaries.
  • You can simultaneously use multiple dictionaries.

Patch Analyzer

You can sort results by SQL Server name and a further sort by column within each group.

Quick Reindex

  • Index results are no longer automatically resorted after re-indexing.
  • You have the option to turn off the re-indexing warning message.
  • You can select individual databases before loading index statistics.

Server Configuration

  • Configuration information now includes service pack level, server collation, and login audit information.
  • Comparison results now display differences in server collation values. You cannot update these differences.

Server Ping

You now can start, stop, pause, resume, and restart your SQL Server instances using the Server Ping tool.

Server Statistics

  • The server state is now correct for SQL Server 2000.
  • The results for per-second counters (e.g. logins/sec) are more accurate.

Space Analyzer

  • The Space Analyzer tool now has the ability to display space information for local mount points. However, it only detects mount points on Windows 2003, 2008, and Vista, and the mount point must host a valid database.
  • Results now include system databases (e.g. tempdb, msdb).
  • Results include a new column % Disk Used by Database File showing how much disk space is used by the selected file.

SQL Discovery

When scanning, you may enter a range of IP addresses or a local subnet.

User Clone

  • The Default Database field for a new user correctly displays databases from the destination SQL Server.
  • Source User displays all Windows authenticated group logins.
  • You have the option to exclude role-level object permissions.
