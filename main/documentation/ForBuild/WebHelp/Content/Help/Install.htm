How to install

Install SQL admin toolset on your computer or workstation. You can also install the SQL admin toolset on the SQL Server computer hosting the databases you want to manage with these tools.

To install the SQL admin toolset:

  1. Review the product requirements.
  2. Log in to the target computer using an account that has administrator permissions.
  3. Download and install the SQL admin toolset.
  4. From the Tools available for install list, select SQL admin toolset.
  5. Click Install.
