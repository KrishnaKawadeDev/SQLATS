Access the SQL admin toolset Help

While the Release Notes provide you with the most current information regarding this SQL admin toolset release, the main Help gives you the detail you need to understand SQL admin toolset and how it provides many different tools to diagnose, administer, and analyze SQL Server instances across your enterprise.

You can use the main SQL admin toolset Help to:

  • Verify the backup status of your SQL Server instances
  • Checking connections to your SQL Server instances
  • Move databases between SQL Server instances
  • Determine the health of your SQL Server instances
  • Manage your SQL Server patches
