Configure authentication

Each tool requires a valid Windows user account or SQL Server login credentials to connect to your SQL Server instances and databases. By default, the SQL admin toolset uses the credentials of your currently logged-on Windows account.

You can configure the authentication method and credentials in the following ways:

  • When using any tool, click the user access ( ) icon.
  • When managing a server group, click Set Server Credentials.

These options let you enforce authentication standards and corporate security policies across your server groups while providing the ability to use alternate credentials on an ad-hoc basis to address specific connection needs.
